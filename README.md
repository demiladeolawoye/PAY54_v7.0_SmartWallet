# PAY54 v7.0 — BASELINE FREEZE

Status: LOCKED  
Version: v7.0  
Date: 14 December 2025

## Purpose
This repository contains the frozen frontend baseline for PAY54 v7.0.
It is a functional UI build intended for demos, investor review, and backend integration.

## Scope included (v7.0)
- Auth screens (Login, Signup, OTP, Recover, Reset PIN)
- Session guard (frontend only)
- Light / Dark theme with user toggle
- Official PAY54 branding (v6.7-aligned logos)
- Dashboard UI
- ONE wallet, multi-currency (NGN, USD, GBP, EUR)
- Send, Add, Withdraw (mock)
- PIN re-authentication
- Transaction history (mock)
- Receipt modal + WhatsApp share
- Cross-border FX mock (“You send / They receive”)

## Scope excluded (NOT part of v7.0)
- Backend APIs
- Real authentication
- Real wallet ledger
- Savings, Bills, Investments
- Agents / Agent onboarding
- AI Risk Watch
- Real FX rates
- KYC / AML
- Compliance flows
- Production security hardening

## Folder notes
- Some folders/files exist for future or experimental work.
- Presence of a file does NOT imply it is active or in scope.
- Only items listed under “Scope included” define PAY54 v7.0.

## Rules
- No feature additions in v7.0
- Bug fixes only
- All new features must target v7.1+
